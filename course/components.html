Use .slide__grid inside .slide__content. Column classes span from 1 to 12. Named aliases follow PDS area language.


At 1920×920px: column ≈ 113.7px, gap = 36px. Combined: 12×113.7 + 11×36 = 1760px = full content width.

<pc-stat-counter>
Animated number counter with accent fill bar. Triggers on scroll into view. GSAP or rAF fallback.
Default — percentage
Custom suffix + max
No suffix
Decimal
<pc-stat-counter
  value="94"
  label="Customer Satisfaction">
</pc-stat-counter>

<pc-stat-counter
  value="630"
  suffix="K+"
  label="Avg. Household Income"
  max="1000"
  duration="2.5">
</pc-stat-counter>

<pc-stat-counter
  value="34"
  suffix=""
  label="Service Quality (CSI)">
</pc-stat-counter>

// Attributes
// value     target number          required
// label     text below number      optional
// suffix    appended to number     default: "%"
// max       denominator for bar    default: 100
// duration  animation seconds      default: 2
// delay     seconds before start   default: 0
<pc-card>
Interactive explore card. Three states: locked (shimmer), unlocked (clickable), revealed. Used in card-explore slide layouts.
<pc-card
  label="Design & Function"
  body="Every Porsche is engineered
        to be as beautiful as it is
        functional.">
</pc-card>

// Attributes
// label   card title text    required
// body    revealed content   required
// icon    path to icon img   optional

// JS API
card.unlock()    // enable clicking
card.reveal()    // show body content
card.lock()      // reset to shimmer
card.isRevealed  // boolean

// Events
card.addEventListener(
  'pc-card-revealed',
  () => checkAllVisited()
);
<pc-objective-list>
Numbered learning objectives. Reveal one at a time in sync with VO, or all at once with stagger.
<pc-objective-list
  items="First objective|
         Second objective|
         Third objective">
</pc-objective-list>

// Pipe-separate items in the
// items attribute.

// JS API
list.revealNext()
// Reveals next hidden item.
// Returns true if more remain.

list.revealAll()
// Reveals all with 150ms stagger.

list.reset()
// Hides all items.

// Properties
list.revealedCount  // number
list.totalCount     // number

// Events
list.addEventListener(
  'pc-objectives-complete',
  () => unlockNext()
);

<pc-bar-chart> + <pc-bar>
Animated horizontal bar chart. Bars stagger in on scroll. One bar can be highlighted in accent red to emphasize a key metric.
With highlight — CSI dimensions
No highlight — equal weight
<pc-bar-chart unit="%" max="40">
  <pc-bar
    label="Service Quality"
    value="34"
    highlight>
  </pc-bar>
  <pc-bar
    label="Service Advisor"
    value="22">
  </pc-bar>
</pc-bar-chart>

// Attributes on <pc-bar-chart>
// unit      value suffix      default: "%"
// max       100% bar width    default: 100
// duration  seconds/bar       default: 1.5
// delay     start delay (s)   default: 0

// Attributes on <pc-bar>
// label     row label         required
// value     numeric value     required
// highlight accent red bar    boolean

// Events
chart.addEventListener(
  'pc-bars-complete',
  () => console.log('done')
);
<pc-callout>
Inline highlight box for quotes, key insights, and callouts within slide content. Supports rich inner HTML.
accent (default)
Porsche customers are buying brand experience, not just a vehicle type.
quote
Porsche service has to be different because Porsche is different.
neutral
Your technical communication quality directly impacts the 34% of CSI that you control.
info
Porsche has ranked #1 in J.D. Power Customer Appeal for 10+ consecutive years. 
<pc-callout
  label="Key Insight"
  variant="accent">
  Content with <strong>bold</strong>
  supported.
</pc-callout>

// Attributes
// label     heading text    optional
// variant   visual style    default: "accent"

// variant options:
// "accent"
//   ! icon, PDS error-soft bg (#3A0F0F)
//   → key takeaways, important notes

// "info"
//   i icon, PDS info-soft bg (#04294E)
//   → facts, data, statistics

// "neutral"
//   — icon, dark surface bg
//   → "what this means" notes

// "quote"
//   " icon, dark surface bg, italic
//   → attributed quotes

// Inner HTML is preserved —
// place content directly inside.
